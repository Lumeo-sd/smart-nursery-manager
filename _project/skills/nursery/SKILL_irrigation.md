# SKILL: OpenSprinkler — Керування поливом

**Коли:** "полий зону A", "стан поливу", "зупини полив", "затримка через дощ"
**Статус:** MCP підключений, потребує налаштування IP контролера

---

## Контролер

OpenSprinkler — фізичний WiFi-контролер поливу на розсаднику.

| Параметр | Значення |
|---|---|
| `OPEN_SPRINKLER_HOST` | IP контролера (налаштувати в .env) |
| `OPEN_SPRINKLER_PASSWORD` | Пароль контролера |

---

## Доступні команди

| Команда | Що робить |
|---|---|
| `get_controller_status` | Стан контролера, час, rain delay |
| `get_stations` | Всі зони з поточним статусом |
| `run_station` | Запустити зону на N хвилин |
| `stop_station` | Зупинити конкретну зону |
| `stop_all_stations` | Зупинити все негайно |
| `set_rain_delay` | Затримка через дощ (годин, 0 = скинути) |
| `get_programs` | Розклад автоматичного поливу |
| `enable_controller` | Увімкнути/вимкнути контролер |

---

## Інтеграція з розсадником

**Сценарії використання:**

- Після запису Batch Event "Полив" → підтвердити що зона поливалась
- Перевірити стан перед ранковим обходом
- Екстрена зупинка при поломці
- Встановити rain delay після дощу автоматично (+ weather API в майбутньому)

---

## Статус підключення

⚠️ Потребує налаштування: додати IP і пароль контролера в конфіг Claude Desktop.
Коли буде налаштовано — можна керувати поливом прямо з чату.
