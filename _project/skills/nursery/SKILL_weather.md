# SKILL: Weather MCP — Погода для розсадника

**Статус: ✅ АКТИВНИЙ — Open-Meteo API (безкоштовно, без ключа)**
**Модуль:** `mcp/weather/` | Python пакет: `mcp-weather-server`

---

## Що вміє

| Інструмент | Дані |
|---|---|
| `get_current_weather` | Температура, вологість, вітер, опади, UV, "відчувається як" |
| `get_weather_byDateTimeRange` | Погодинний прогноз на N днів |
| `get_air_quality` | PM2.5, PM10, озон — якість повітря |
| `get_current_datetime` | Поточний час у будь-якій зоні |
| `convert_time` | Конвертація між зонами |

---

## Використання в розсаднику

### Ранковий дайджест (додати до SKILL_status)
```
get_current_weather(city="Ваше місто, Ukraine")
→ показати: температура, чи очікується дощ, UV
→ якщо temp < 2°C → ⚠️ МОРОЗ: перевір теплицю!
→ якщо precipitation > 5mm → 💧 Полив пропустити
```

### Тригери для розсадника

| Умова | Дія |
|---|---|
| Температура < 0°C вночі | ⚠️ Закрити Теплицю А, перемістити розсаду |
| Температура < +2°C (навесні) | ⚠️ Критично для живців — накрити |
| Температура > 30°C | 💧 Збільшити частоту поливу |
| Опади > 10mm | ⏭️ Пропустити полив OpenSprinkler |
| UV індекс > 7 | 🌂 Затінення для ніжних рослин |

### Приклад запиту
```python
get_current_weather(city="Kyiv, Ukraine")
# або вказати точніше місто розсадника
```

---

## Інтеграція з OpenSprinkler

```
Weather: precipitation_probability > 60%
  → OpenSprinkler: пропустити полив
  → Batch Event: "Полив пропущено — дощ прогноз X%"
```
