# SKILL: Weather MCP — Погода для розсадника

**Статус: 🚧 Заглушка — реальний API в розробці**
**Модуль:** `mcp/weather/`

---

## Що робить (після реалізації)

| Інструмент | Коли використовую |
|---|---|
| `get_current_weather` | Ранковий дайджест — додати до SKILL_status |
| `get_frost_alert` | ⚠️ КРИТИЧНО — мороз → дія сьогодні вночі |
| `get_irrigation_recommendation` | Перед OpenSprinkler — чи поливати? |
| `get_forecast` | Планування на тиждень |

---

## Інтеграція в ранковий дайджест

```
🌱 ДАЙДЖЕСТ — [дата] [час]

🌤️ ПОГОДА (Kyiv +2):
  Зараз: +8°C, хмарно, вітер 12 км/г
  Сьогодні: +4°C...+11°C
  ⚠️ УВАГА: Вночі очікується -1°C → перемістити розсаду!

💧 ПОЛИВ: пропустити (завтра дощ 70%)

📦 ПАРТІЇ: ...
```

---

## Сезонні тригери (Україна)

| Подія | Температура | Дія |
|---|---|---|
| Мороз | < 0°C | Закрити теплицю, сповістити |
| Спека | > 30°C | Збільшити полив, затінення |
| Дощ | > 60% | Пропустити полив |
| Заморозки навесні | < +2°C вночі | ⚠️ Критично для живців |
