# SKILL: Nursery Status — Дайджест системи

**Коли використовувати:** "що зараз?", "покажи стан", "дайджест", "що є в наявності"

---

## Алгоритм виконання

### 1. Збери дані (паралельно)
```
→ erpnext_run_method: Plant Batch список (всі активні)
→ erpnext_get_account_balance: Cash - SDR
→ erpnext_run_method: Sales Invoice count за сьогодні
```

### 2. Структура відповіді

```
🌱 ДАЙДЖЕСТ РОЗСАДНИКА — [дата]

📦 ПАРТІЇ (N активних):
  ✅ Готові до продажу:
     • [назва] — [кількість] шт @ [ціна] грн (Теплиця/Майданчик)
  🌿 В роботі:
     • [назва] — [кількість] шт — [статус] — готовність: [дата]
  ⚠️  Потребують уваги: (якщо є втрати або прострочені)

💰 ФІНАНСИ:
  • Каса: [сума] грн
  • Продажів сьогодні: [N] шт на [сума] грн

⚡ ПОТРІБНА УВАГА: (тільки якщо є)
  • [конкретна дія]
```

### 3. Правила форматування
- Якщо партій немає — написати "Система порожня"
- Якщо каса = 0 — написати "Каса порожня (жодного продажу)"
- Прострочена дата готовності = статус "В роботі" але `expected_ready_date` < сьогодні → виділити ⚠️

---

## Поля Plant Batch що нас цікавлять
```
batch_title, plant_variety, pot_size, status, quantity_current,
quantity_lost, cost_per_plant, expected_ready_date, current_location
```

## Статуси партій
- `Готова до продажу` → показуємо в "Готові"
- `Укорінення`, `Зростання`, `Живцювання`, `Готова до пересадки` → "В роботі"
- `Закрита`, `Втрачена` → не показуємо
