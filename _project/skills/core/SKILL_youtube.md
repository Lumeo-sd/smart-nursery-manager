# SKILL: YouTube Transcript MCP

**Коли:** "що кажуть в цьому відео", "зроби конспект", "знайди момент де..."
**Модуль:** `youtube_transcript` (Docker: mcp/youtube-transcript)

---

## Що робить

Отримує повний текстовий транскрипт будь-якого YouTube відео за URL або ID.

## Для проекту розсадника — застосування

| Ситуація | Як використати |
|---|---|
| Навчальне відео по агротехніці | Зробити конспект → зберегти в `docs/agro/` |
| Відео конкурента про їх асортимент | Проаналізувати без перегляду |
| Туторіал по ERPNext/React | Витягти кроки → додати в документацію |
| Вебінар по бізнесу розсадника | Конспект + ключові ідеї |

## Приклад
```
"Зроби конспект цього відео: https://youtube.com/watch?v=XXX"
→ отримую транскрипт → структурую → зберігаю в docs/
```
