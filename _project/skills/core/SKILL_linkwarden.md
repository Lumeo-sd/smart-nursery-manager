# SKILL: Linkwarden MCP — Менеджер закладок

**Коли:** зберегти посилання, знайти збережену сторінку, організувати ресурси
**Модуль:** `linkwarden` (Docker: linkwarden_linkwarden_net)

---

## Що робить

Linkwarden — self-hosted менеджер закладок. Зберігає посилання з тегами і колекціями.

## Для проекту — застосування

| Тип ресурсу | Колекція |
|---|---|
| ERPNext документація | `erpnext-docs` |
| Агрономічні статті | `agronomy` |
| React / PWA туторіали | `dev-pwa` |
| Конкуренти і ринок | `market-research` |
| Постачальники субстрату | `suppliers` |

## Коли використовую в сесії

- Знайшов корисну статтю → зберігаю через Linkwarden замість губити в чаті
- Потрібен ресурс → шукаю в збережених замість Google
- Дослідження ринку → зберігаю сайти конкурентів для подальшого аналізу
